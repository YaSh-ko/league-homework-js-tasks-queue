# league-homework-js-tasks-queue

## Эта работа базово демонстрирует работу EventLoop

Порядок выполнения операций:

1. Задача 1 (синхронный код)
   - Вывод сообщения "task 1"
   - Выполнение микротаски, вывод сообщения "microtask 1"
   - Рендер браузером измененных стилей

2. Задача 2 (таймер)
   - Вывод сообщения "task 2"
   - Выполнение микротаски, вывод сообщения "microtask 2"
   - Выполнение микротаски, вывод сообщения "another microtask 2"

3. Задача 3 (еще один таймер)
   - Вывод сообщения "task 3"
   - Выполнение микротаски, вывод сообщения "microtask 3"
   - Рендер браузером измененного контента
